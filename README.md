# F1 Race Visualization
A Python application for visualizing Formula 1 race telemetry and replaying race events with interactive controls and a graphical interface.

### Requirements
These are the required dependencies and the libraries in order to this app.
```
Python 3.11+
FastF1
Arcade
numpy
```

### Install dependencies
Install all of them by running one command below, All requirements have named aliases in the requirements.txt file.

```
pip install -r requirements.txt
```


### Run it
Boooooyaaaaah, Run it now and see it in action.

```
python main.py
```
